# 🧠 MAYAM Architecture (Simple View)

User Input (Email / Link / Message)
                ↓
        TRAE Agent (Module 1)
     (Data Collection Layer)
                ↓
        MAYAM Engine (Module 2)
     (AI Intelligence Layer)
   ┌──────────────────────────────┐
   │ • Intent Detection           │
   │ • Link Analysis              │
   │ • Sender Analysis            │
   │ • Deception (LLM)            │
   │ • Risk Scoring               │
   │ • Guidance Generation        │
   └──────────────────────────────┘
                ↓
        Dashboard (Module 3)
     (User Interaction Layer)
   ┌──────────────────────────────┐
   │ • Risk Score                 │
   │ • Tactics Detected           │
   │ • Explanation                │
   │ • Safe Reply Suggestion      │
   │ • Next Attack Prediction     │
   └──────────────────────────────┘

⚡ Core Idea:
MAYAM doesn’t just detect threats — it guides user decisions in real-time.
